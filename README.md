# 📦 Product Inventory Management System

## 📌 Description
### A menu-driven console-based Product Inventory Management System developed in Python.
This application allows users to manage product records efficiently using JSON file storage.

The system supports adding, updating, deleting, searching, and viewing products with automatic data persistence.

## 🚀 Features
• ➕ Add new products

• 🔄 Update product quantity and price

• ❌ Delete products

• 🔍 Search product by ID

• 📦 View complete inventory

• ⚠️ Low stock alert system

• 💾 Data stored permanently in inventory_data.json

## 🛠 Technologies Used
• Python 

• JSON (for data storage)

• OOP (Object-Oriented Programming)

• File Handling
